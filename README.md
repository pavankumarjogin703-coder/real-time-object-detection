# 🚀 Real-Time Object Detection System

A professional AI-powered real-time object detection system built using YOLOv8, OpenCV, and Python.

This application detects objects directly from webcam feed with live FPS monitoring, object counters, colored bounding boxes, and screenshot saving.

---

# 📸 Features

✅ Real-time webcam object detection  
✅ YOLOv8 AI model  
✅ FPS counter  
✅ Colored bounding boxes  
✅ Object counters  
✅ Screenshot saving  
✅ Custom object filtering  

---

# 🧠 Detected Objects

- 👤 Person
- 📱 Cell Phone
- 🚗 Vehicles
- 🍼 Bottle

# ⚙️ Installation

## Clone Repository

```bash
git clone https://github.com/YOUR_USERNAME/real-time-object-detection.git
```

## Open Project Folder

```bash
cd real-time-object-detection
```

## Create Virtual Environment

```bash
python3 -m venv venv
```

## Activate Virtual Environment

```bash
source venv/bin/activate
```

## Install Dependencies

```bash
pip install -r requirements.txt
```

---

# ▶️ Run Project

```bash
python3 main.py
```
